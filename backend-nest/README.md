# Warehouse Backend (NestJS)

## Quick Start
1. npm install
2. npm run start:dev

Server default:
- http://localhost:5001
- API prefix: /api

Health:
- GET /api/health

Default bootstrap admin (created automatically):
- username: admin
- password: password123

## Implemented Modules
- auth
- employees
- devices
- health
